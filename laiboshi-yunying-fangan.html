一页会议纪要

结论一句话：赖博士养鸭走真人博士IP路线，用SABC生产法"每月集中拍2天、囤够一个月素材"，视频号为主战场，前期建信任涨粉、第3个月出转化。

1. 公司与账号

龙岩曼纽考（母公司新奥农牧，27年脂质营养）旗下品牌"欧米伽博士"。产品：DHA富集蛋、连城白鸭、水培菜。核心卖点：DHA富集（蛋黄三文鱼纹、水煮无腥）、全程无抗NAE、立体生态净养、连城白鸭国家地理标志/元代药膳。创始人赖州文博士。

2. 核心判断

  • 现有内容"三层混讲"（技术/鸭肉/鸭蛋），对外应聚焦DHA鸭蛋一个主角，技术和博士故事当信任背书。
  • 人设：返乡实干博士 + 把硬核技术讲成大白话。保留客家情怀和真实，啰嗦靠后期剪，不强迫博士改说话方式。
  • 对标杨博士（博士返乡养鸡，学模式不照搬）。

3. 关键决策

  • 呈现：真人为主（博士第一人称导游为魂），暂不做直播。
  • 生产：SABC框架法——S类畅讲、A类采访、B类拆爆款，每月拍2天供多号发一个月。
  • 近期动作：去仙坪基地5-10天，趁博士在，一次囤够素材。

4. 待确认

账号具体目标（品牌/获客权重）、赖博士出镜配合度与固定时间、现有客户画像、DHA倍数等数据口径（找QA虞业凌核准）。

下一步：①按基地拍摄清单出差囤素材 → ②回来剪第一条（博士回乡）→ ③按节奏起号。

总策略：3个月主线

第0步
基地囤素材（出差5-10天）
趁老板赖博士在，一次拍够1-2个月弹药。盯：博士人物素材够没。
第1月
摸清+起号：了解产品/走鸭场/拿数据 → 定定位人设内容日历 → 发5-10条测试 → 找爆款规律。盯：号立住没。
第2月
跑内容+涨粉：复制跑通的方向批量做，一鱼多吃分发。盯：视频号涨粉、单条互动。
第3月
转化变现：开小黄车、做带货向内容。盯：咨询量、转化订单。
每步只盯一个核心指标，不贪多。别一上来就想变现，那样会崩。

保留 / 优化

保留

优化
啰嗦、专业词多、开头铺垫长——用剪辑、字幕、重点提炼后期救，不强迫他改。

SABC 生产法（核心）

不追数量追目的。每条要么S又爆又转、要么A有转化、要么B拉权重大爆款，没有垃圾素材。

类目的生产模式博士怎么拍月产
S又爆又转猜结构+自由长讲+后期重组只给一个选题，畅讲20-30分钟4条
A精准转化拆框架采访式给开头+金句+结尾，中间自由发挥，拍10分钟12条
B纯流量拆爆款写逐字稿提词器念/员工念，情绪足节奏快30秒内5条

S 4条 · 又爆又转（畅讲后期重组）

  • S1《我养了这么多年鸭，说句得罪同行的大实话》— 内幕爆点→落无抗/圈纹
  • S2《一个博士为什么放着好日子不过回村养鸭》— 情感故事破圈+立信任
  • S3《为什么很多人买了高价鸭蛋会后悔？3点》— 教避坑+带出自家标准
  • S4《30岁后越来越在意吃什么，我用一颗蛋讲明白》— 健康焦虑+给家人吃

A 12条 · 精准转化（采访框架）

  • 孩子能吃：A1 鱼油还是鸭蛋 / A2 几岁能吃
  • 值不值：A3 凭什么贵 / A4 几块和十几块差在哪
  • 真不真：A5 无抗拿报告说话 / A6 鸭场臭不臭
  • 怎么吃：A7 水煮就够 / A8 清水炖鸭汤
  • 送礼健身通用：A9 送长辈 / A10 健身蛋白 / A11 立体生态大白话 / A12 药膳鸭由来

B 5条 · 纯流量（拆爆款写稿）

  • B1 博士不当教授回村养鸭图啥 / B2 6万只鸭一个人管 / B3 鸭粪变成菜 / B4 蛋剥开像三文鱼 / B5 鸭子喝西北风也能长
涉及DHA倍数、无抗年限等数据，发布前找QA虞业凌核准；药膳是文化说法，不能延伸成治病功效。

开头留人公式

第一句别说自己，说观众关心的事。前3秒抛钩子，不铺垫。

痛点提问
"想给孩子补DHA，是不是在鱼油和保健品里挑花了眼？"
颠覆认知
"贵的鸭蛋不一定好，但好的鸭蛋一定不便宜——差在哪你得懂。"
争议+后果+方法
"都说养鸭没技术含量，可我一个博士在这栽了七年。"
大众以为+反差+原因
"一说养鸭场你先想到臭味？我这不一样，原因有三个。"
反差悬念
"二十二年前走出大山，所有人以为我不回来了。"
视觉冲击
"（切开鸭蛋）你看这蛋黄的圈纹，这是有讲究的。"
